# Vlessbot2

Telegram bot for managing VLESS subscriptions with a referral system and payment integration.

## Features

- User and Admin panels
- Subscription management (tariffs, servers)
- Two-tier referral program
- Payment integration (YooKassa, Telegram Stars)
- Automated VPN key generation via X-UI panel
- Dockerized for easy deployment
- Database migrations managed by Alembic.
CI/CD is configured with GitHub Actions for automated deployments. This simplifies the update process.
